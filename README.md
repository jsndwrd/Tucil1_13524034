# Queens

### IF2211 Algorithm Strategy Mini Project I

LinkedIn Queens puzzle solver.

## About Program

This program is made for IF2211 Algorithm Strategy course mini project submission at ITB Informatics Engineering. The program is built using Go to search solution for LinkedIn Queens game using pure Brute Force approach.

## Instructions

1. Install Go on your machine
2. Use one of these commands to compile / run the program
   - Compile & run

     ```
     cd src
     go build -o ../bin/queens.exe .
     ../bin/queens.exe
     ```

   - Run directly
     ```
     cd src
     go run .
     ```
